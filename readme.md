## 常用指令
- [ ] `go`
	- `go mod init <模块名>`: 初始化模块。
	- `go mod tidy`: 同步模块。
- [ ] `redis`
	- `redis-server`: 开启 `redis` 服务器。
	- `redis-cli -h 127.0.0.1 -p 6379`: 连接 `redis` 服务器。
	- `keys *`: 查看 `redis` 服务器的所有键。
